AvaTax MCP Server

Tax calculation and address validation for Claude Desktop • Powered by Avalara

Welcome to AvaTax MCP Server!

Integrate Avalara's industry-leading tax calculation and compliance platform with Claude Desktop through the Model Context Protocol (MCP). Access real-time tax calculations, address validation, and transaction management directly from your AI conversations.

🧮 Precise Tax Calculation

Calculate sales tax, VAT, GST, and other transaction taxes with line-item detail using Avalara's comprehensive tax engine trusted by 40,000+ businesses worldwide.

📍 Global Address Validation

Validate and normalize addresses across 75+ countries for accurate tax calculations and compliance requirements.

📊 Transaction Management

Create committed transactions directly in AvaTax for seamless integration with your existing tax workflow and audit requirements.

🌍 Multi-Environment Support

Seamlessly switch between Avalara's sandbox and production environments for testing and live operations.

🔌 Claude Desktop Integration

Connect the power of Avalara's tax automation platform to your AI workflow:

  1. Configure your AvaTax credentials in the Configuration tab
  2. Copy the auto-generated Claude configuration (appears after saving credentials)
  3. Add it to your Claude Desktop configuration file
  4. Restart Claude Desktop to activate the integration

✨ Smart Configuration: Your personalized Claude Desktop configuration is automatically generated based on your credentials - no manual JSON editing required!

🚀 Instant Access: The application launches automatically after installation and upgrades, ensuring quick access to your tax compliance tools.

📚 Documentation

                    
                    
Claude Desktop Configuration

                    
✨ For your personalized configuration, go to the Configuration tab and save your AvaTax credentials. The exact configuration you need will be generated automatically!

                    
                    
The generated configuration needs to be added to your claude_desktop_config.json file. Here's the general structure:

                    
                    
{
  "mcpServers": {
    "avatax": {
      "command": "node",
      "args": ["path/to/app.asar/dist/index.js"],
      "env": {
        "AVATAX_ACCOUNT_ID": "your_actual_account_id",
        "AVATAX_LICENSE_KEY": "your_actual_license_key", 
        "AVATAX_COMPANY_CODE": "your_actual_company_code",
        "AVATAX_ENVIRONMENT": "sandbox_or_production",
        "AVATAX_APP_NAME": "AvaTax-MCP-Server",
        "AVATAX_TIMEOUT": "30000"
      }
    }
  }
}

                    
                    

                        
📍 Finding claude_desktop_config.json

                        
The configuration file is located at:

                        
    
                            
  • Windows: %APPDATA%\\Claude\\claude_desktop_config.json
    • 
                            
  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
    • 
                            
  • Linux: ~/.config/Claude/claude_desktop_config.json
    •
