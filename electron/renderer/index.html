This application provides AvaTax integration for Claude Desktop through the Model Context Protocol (MCP).

üìç Address Validation

üîß Multi-Environment

üîå Claude Desktop Integration

To use this server with Claude Desktop, you'll need to:

  1. Configure your AvaTax credentials in the Configuration tab
  2. Start the MCP server in the Server Control tab
  3. Add the server to Claude Desktop configuration (instructions in Documentation tab)

Note: The server runs locally and connects Claude Desktop to your AvaTax account securely.

üöÄ Quick Start

New to AvaTax? Here's what you need:

  • An AvaTax account (sign up at avalara.com)
  • Your Account ID and License Key from the AvaTax portal
  • Your Company Code (found in AvaTax under Companies > Company Settings)
  • Claude Desktop installed on your computer

AvaTax Configuration

Configure your AvaTax credentials and settings. These will be saved securely on your computer.

MCP Server Control

Start and stop the AvaTax MCP server for Claude Desktop integration.

üìö Documentation

Claude Desktop Configuration

To connect this MCP server to Claude Desktop, add the following to your claude_desktop_config.json file:

{
  "mcpServers": {
    "avatax": {
      "command": "node",
      "args": ["C:\\Users\\YourUsername\\AppData\\Local\\Programs\\AvaTax MCP Server\\resources\\app.asar\\dist\\index.js"],
      "env": {
        "AVATAX_ACCOUNT_ID": "your_account_id",
        "AVATAX_LICENSE_KEY": "your_license_key", 
        "AVATAX_COMPANY_CODE": "your_company_code",
        "AVATAX_ENVIRONMENT": "sandbox"
      }
    }
  }
}

üìç Finding claude_desktop_config.json

The configuration file is located at:

  • Windows: %APPDATA%\\Claude\\claude_desktop_config.json
  • macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
  • Linux: ~/.config/Claude/claude_desktop_config.json

Available Commands

Once connected, you can ask Claude to:

calculate_tax

"Calculate sales tax for a $100 item shipped from California to New York"

validate_address

"Validate this address: 123 Main St, Seattle, WA 98101"

create_transaction

"Create a committed sales transaction for order #12345"

Getting AvaTax Credentials

  1. Sign up for an AvaTax account at avalara.com
  2. Access the AvaTax portal and navigate to Settings > License Keys
  3. Copy your Account ID and License Key
  4. Find your Company Code under Companies > Company Settings
  5. Start with sandbox environment for testing
